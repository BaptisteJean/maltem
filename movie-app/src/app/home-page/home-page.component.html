Welcome to Movie Library Assignment


Movie Library Assignment is a movie management application. The goal of this project is to evaluate my technical skills as java full stack developper and other technologies such as Angular Version 6, Spring framework Version 2.5.1, Bootstrapp Version 4, Java Version 8, Json, Git (the basics), etc.

In this application we will be able to use some crud methods such as:
  1. The creation of a movie,
  2. The removal of a film,
  3. Updating a movie
  4. Filter movies according to a research element etc
The project is subdivided into two folders:
  1. movielibraryassignment that contains all the backend side of the application,
  2. movie-app which is the client side using the restServices put together in 1

The way to compile/build/run it


  1. you must first clone the github repository in a folder of your machine,
  2. then go to the folder movielibraryassignment "cd movielibraryassignment",
  3. then execute the command "mvn spring-boot: run",
  4. then go back to the main directory,
  5. go to the movie-app folder "cd movie-app" do "ng serve --open" or "ng serve",
  6. open an internet browser and enter the link http: // localhost: 4200
